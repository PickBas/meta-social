Автроизация

Автроизация

Если вы уже зарегистрировались, введите свой логин и пароль.
Либо войдите через один из трех сервисов (Яндекс, Вконтакте, Facebook).

Регистрация

Для регистрации укажите все данные, затем войдите используя их.

Восстановление пароля

Если вы забыли свой пароль нажмите на кнопу "Забыли пароль?".
Далее нужно указать свою электронную почту от аккаунта, на неё придет сообщение с ссылкой для смены пароля. Перейдя по ней придумайте новый пароль и укажите его.

Профиль

Основное

Вы можете просматривать профиль каждого пользователя Meta Social, а так-же его друзей и т.д.
Просмотр друзей
Для просмотра друзей пользователя нажмите на эту кнопку.
Просмотр сообществ
Для просмотра сообществ пользователя нажмите на эту кнопку.
Просмотр музыки
Для просмотра музыки пользователя нажмите на эту кнопку.

Редактирование

Редактирование информации
Для редактирования профиля нажмите на кнопку под вашей аватаркой.
Смена аватарки
Для смены аватарки нажмите на аватарку в своем профиле.

Сообщения

Создание чата

С одним человеком
Чтоб начать переписку с другом, перейдите во вкладку "Друзья", затем нажмите на кнопку "TEXT".
С несколькими людьми
Чтоб создать беседу нажмите на данную кнопку.

Отправка сообщений

Перед отправкой сообщений вы можете выбрать изображения которые хотите отправить нажав на скрепку.
Так-же сообщения можно отправлять нажатием enter.

Друзья

Поиск

Для поиска друзей просто начните вводить имя или никнейм друга.

Управление

В меню поиска, заявок и черного списка, у вас есть возможность выполнять некоторые действия нажатием на кнопки справа. Чтобы узнать что выполняет данная кнопка задержите на ней курсор пока не всплывет подсказка.

Сообщества

Поиск

Для поиска сообщества просто начните вводить его название.

Создание сообщества

Редактирование информации
Для редактирования информации о сообществе нажмите на соответствующую кнопку под аватаркой сообщества.
Смена аватарки
Для смены аватарки сообщества нажмите на неё.
Управление подписчиками
Вы можете выдавать и забирать привелегии пользователей в списке подписчиков сообщества.

Музыка

Добавление музыки

Загрузка новой музыки
Выберите песню которую хотите загрузить, и введите автора и название.
Добавление существующей
Просто начните вводить автора или название песни.

Посты

Управление

Если вам понравилась запись нажмите на сердечко снизу. Так-же вы можете высказать свое мнение о записи в комментарии снизу.
Если вы хотите поделиться этой записью со своими друзьями нажмите на кнопку с громкоговорителем, запись появится у вас в профиле.
Чтоб удалить/редактировать/открыть запись нажмите на 3 точки справа вверху.

Редактирование

Для смены позиции картинок, просто перетаскивайте их.
